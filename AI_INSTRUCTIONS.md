# AI_INSTRUCTIONS.md

## Mission

Ce projet est un site d'affiliation Amazon spécialisé dans :

1. L'électroménager
2. Les produits pour chiens et chats

L'utilisateur fournit uniquement un ou plusieurs mots-clés.

Exemples :
* aspirateur robot
* machine à café automatique
* croquettes pour chien
* fontaine à eau pour chat

À partir de ces mots-clés, l'IA doit réaliser tout le reste de manière autonome.

---

## Workflow obligatoire

### 1. Recherche Internet
* Effectuer des recherches sur Internet à partir du mot-clé fourni.
* Identifier les produits réellement liés au mot-clé.
* Utiliser des sources fiables : fabricants, tests d'experts, comparatifs reconnus et avis vérifiés.
* Ne jamais inventer de produits.

### 2. Sélection des produits
* Sélectionner les 6 produits les plus pertinents.
* Éviter les produits obsolètes ou introuvables.
* Préférer les produits populaires et bien notés.

### 3. Informations à récupérer
Pour chaque produit :
* Nom complet
* Marque
* Images du produit provenant de sources autorisées
* Principales caractéristiques
* Avantages
* Inconvénients
* Utilisateur recommandé
* Note globale

### 4. Création de l'article
Générer automatiquement un article complet :

# Les 6 meilleurs [mot-clé] en [année]

Sections obligatoires :
1. Introduction
2. Tableau comparatif avec images
3. Fiche détaillée des 6 produits
4. Guide d'achat
5. Questions fréquentes (FAQ)
6. Conclusion :
   * Meilleur Choix
   * Meilleur Rapport Qualité/Prix
   * Choix Premium

L'article doit être immédiatement publiable.

### 5. Images
* Ajouter une image pour chaque produit.
* Optimiser les images pour le web.
* Utiliser uniquement des images provenant de sources autorisées.

### 6. Liens d'affiliation Amazon
Tous les liens Amazon doivent utiliser l'identifiant d'affiliation :
StoreID : sababou2026-21

Tous les boutons :
* Voir sur Amazon
* Voir le prix
* Acheter sur Amazon
doivent utiliser automatiquement cet identifiant.

### 7. Respect des règles Amazon
L'IA doit respecter les politiques du programme Amazon Associates :
* Ne pas copier le contenu Amazon.
* Ne pas inventer des avis clients.
* Ne pas afficher des prix Amazon de manière non autorisée.
* Utiliser uniquement des informations fiables et vérifiables.

### 8. SEO Google
L'IA doit produire un contenu optimisé pour le référencement :
* H1 unique
* H2 et H3 structurés
* Introduction utile
* Réponses aux intentions de recherche
* FAQ pertinente
* Maillage interne lorsque possible
* Meta Title
* Meta Description
* URL optimisée
* Données structurées Schema.org (Article, Product et FAQ)

### 9. Comportement attendu
L'IA ne doit jamais demander :
* la liste des produits ;
* les caractéristiques ;
* les images ;
* le contenu de l'article.

L'utilisateur fournit uniquement un ou plusieurs mots-clés.

L'IA doit effectuer elle-même les recherches, générer le comparatif, créer l'article, ajouter les images autorisées, intégrer les liens d'affiliation Amazon avec le StoreID `sababou2026-21` et produire une page prête à être publiée.
