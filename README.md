# Amazon Comparateur

Projet Next.js App Router pour generer des comparatifs de produits a partir de donnees structurees.

## Lancement

```bash
npm install
npm run dev
```

Puis ouvrir:

```text
http://localhost:3000
```

## Ajouter une categorie

Ajoutez une entree dans `components/data/products.ts`. Une page sera generee automatiquement avec le slug.

## Important

- Ne copiez pas le contenu Amazon.
- N'inventez pas les caracteristiques ou avis.
- Remplacez les donnees exemple par des donnees verifiees avant publication.
